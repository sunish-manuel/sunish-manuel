## What I'm Building Right Now

### 🎓 CoreCraft — AI-Powered Adaptive NAPLAN Tutor
> Personalised tutoring for every Australian student, not just those who can afford a human tutor.

An adaptive learning platform for Australian K–9 students aligned to the ACARA V9 curriculum. CoreCraft maps concept-level mastery, identifies exactly where a student's understanding breaks down, and deploys an AI tutor that scaffolds hints progressively rather than revealing answers.

**Stack:** `FastAPI` `React/TypeScript` `PostgreSQL` `TimescaleDB` `Celery` `Redis` `Anthropic Claude API` `Stripe` `Supabase` `Cloudflare Pages` `Railway`

**Key Features:**
- Concept-level mastery engine across 44 curriculum nodes
- Two-gate AI hint mechanism — inline hint first, full tutor on second wrong answer
- Phonics and TTS audio for early literacy components
- Band projection reporting and parent dashboard
- Four-tier SaaS subscription model (Explorer → Apprentice → Master → Forge Academy)

### 📈 AlgoTrade Pro — Multi-Agent Algorithmic Trading Platform
> Institutional-grade infrastructure. Retail-accessible.

An eight-agent LangGraph system where each agent owns a distinct role — signal generation, risk assessment, execution, position monitoring — coordinating through a Redis event bus to make real-time trading decisions across live markets.

**Stack:** `Python` `LangGraph` `LangChain` `Redis` `Groq` `LiteLLM` `Alpaca API` `Interactive Brokers API` `Telegram Bot API`

**Key Features:**
- 8-agent DAG architecture (MARCO, TESLA, FELIX, HAWK, ORACLE, QUINN, REX, GUARDIAN)
- Black-Scholes Greeks engine for options risk (Nifty/BankNifty)
- Three execution lanes: scalp, swing, positional
- Automated risk controls with Telegram live position alerts
- Fully operational in live markets with complete decision audit trail

### 🤖 JOBMASTER — AI-Powered Job Application Automation
> Because job hunting is a workflow problem, not a willpower problem.

An end-to-end AI-powered job application manager that scrapes live postings, generates match and critique scores, tailors ATS-compliant resumes and cover letters, and automates form submissions via browser automation.

**Stack:** `Python` `FastAPI` `React/TypeScript` `PostgreSQL` `Docker` `Playwright` `LangChain` `Anthropic Claude API` `Browser-Use`

**Key Features:**
- Two-pass LLM resume tailoring with Intelligence Package JSON pre-generation
- ATS fingerprinting for Workday, Greenhouse, Lever
- Claude-powered form field reasoning with confidence scoring
- Automated Playwright-based application submission
- 90% reduction in manual job application effort
